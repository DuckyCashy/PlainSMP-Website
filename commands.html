Main Commands

  • /rtp — random teleport
  • /tpa — teleport request
  • /sell — sell items
  • /ah — auction house
  • /rules - views the server rules
  • /sethome — set home
  • /home — teleport home
  • /w - Whisper to Selected User
  • /team - Create a Team, Invite Teammates
Commands may change with updates
